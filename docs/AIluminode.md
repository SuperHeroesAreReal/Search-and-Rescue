# AIluminode

AIluminode is a wieldable posture-before-action orientation instrument.

It is not memory, retrieval, reasoning, moderation, observability, or orchestration. It illuminates contextual posture before an AI or human enters terrain.

Core question:

```text
What terrain am I allowed to enter?
```

## Position

```text
NodAIity tools
├── OhBuoy: continuity pulse
├── Compass: expected vs observed propagation
├── RECCE: bounded field verification
├── Plotter: position instrument
└── AIluminode: posture-before-action orientation instrument
    └── EEG: internal route polarity scan mode
```

Compass determines corridor truth.

AIluminode determines posture before action.

Public identity:

```text
AIluminode = posture-before-action orientation instrument
EEG        = internal scan mode
```

Instrument question:

```text
AIluminode -> How should I proceed?
```

## EEG

EEG is the first AIluminode mode.

Input:

```text
prompt or task
```

Output:

```text
active terrain
stance
Compass guidance
route polarity
drift risk
next safe action
```

Route polarity:

```text
OPEN    = enter this route now
PROTECT = preserve this route; do not alter it
AUDIT   = inspect as evidence before acting
DEFER   = leave dormant unless explicitly reopened
BLOCK   = keep closed for this task
```

Unknown terrain remains outside the route:

```text
unknownTerrain
-> route not registered
-> position uncertain
-> bearing unavailable
-> stop propagation
-> report state
```

## Boundary

AIluminode does not crawl, retain prompts, retrieve memory, ingest source material, modify code, or become an always-on monitor.

It is wielded, read, and released.

```text
illuminate
orient
release
```
